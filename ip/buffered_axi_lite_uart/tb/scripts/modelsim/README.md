ModelSim simulation scripts.
ModelSim is the OEM simulation engine licensed by
multiple FPGA vendors under different names:
  - ModelSim-Intel FPGA Edition (Quartus Lite/Prime)
  - ModelSim-Lattice Edition (Diamond, Radiant)
  - ModelSim-Microchip Edition (Libero SoC)
  - Siemens ModelSim (standalone commercial)
All share the same vsim invocation and do-file syntax.
Questa (scripts/questa/) is the commercial successor
with additional features but compatible syntax.
